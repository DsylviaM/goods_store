# Goods Store

Приложение для управления товарами: загрузка списка, сортировка, добавление и редактирование.

## 🚀 Демо
https://natalia-test-mui.netlify.app

## 🛠️ Стек
- React + TypeScript
- Vite
- CSS Modules

## 📦 Функционал
- Загрузка списка товаров с API
- Сортировка по цене и названию
- Добавление нового товара
- Поиск по названию согласно ТЗ

## 🏃‍♂️ Запуск
npm install
npm run dev
